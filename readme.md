
# Job Application Tracker

A Django application that helps users track job applications,
including status tracking, filtering, search, and dashboard statistics.



## Features

- User authentication
- CRUD for job applications
- Search
- Status filtering
- Pagination
- Dashboard statistics
- Validation rules
- Success messages


## Run Locally

Clone the project

```bash
  git clone https://github.com/ullas-ullas/jobtracker.git
```

Go to the project directory

```bash
  cd jobtracker
```

Install dependencies

```bash
  pip install -r requirements.txt
```

DB setup

```bash
  python manage.py migrate
```

Start the server

```bash
  python manage.py runserver
```
